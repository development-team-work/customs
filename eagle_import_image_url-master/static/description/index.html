Upload Product Images using Image URL

  • This module allow you to upload product images using image url set on product/variant form.

Features:

  • Allow you to set image url and upload image automatically on Save.
  • This module will work for uploading product and product variants images using URL.
  • Supporting Create and Write option. So if you update URL and Edit and Save it read again and set Image.
