Export L/C Management System

This app allow export L/C management workflow of germents business.



This app will work for handel export L/C workflow for garments/textile business/Industry. By this app you can create full set of document of export L/C on pertial delivery challans or full delivery challan.


Main Features:

  • L/C documents related fields are add in Sales Orders
  • There are three pi types ( LOCAL, L/C, L/C Delay).
  • LOCAL type is for local sales order and delivery. LOCAL type has no connecion with L/C
  • L/C type is for add L/C informations with Sales Order.
  • L/C Delay type is for add L/C informations with Sales Order in future.
  • Allow to add signaure of valuable person signature with that sales order.
  • Allow to create proforma invoice PDF on sales oder informations.
  • After confirm order Delivery order will create automatically under factory > Delivery Orders.
  • Allow pertial delivery challans.
  • Delivery Challans will show all done delivery challans.
  • Allow to create delivery challan PDF on delivery challans informations.
  • Allow to create set of documents of Export L/C.
  • Names of list of documents ( Commercial Invoice PDF, Packing List PDF, Bill of Exchange PDF, Truck Challan PDF, Delivery Challan PDF, Delivery/Truck Challan PDF, To Whom PDF, Certificate Of Origin PDF, Quality PDF, Non Uzbek PDF, Forwarding Letter PDF ).
  • Allow to configure L/C Informations.
  • Allow to configure L/C Bank Informations.
  • Allow to configure Beneficiary Bank Informations

    Workflow Overview

    Create a Sales Order. Select PI Type L/C Delay if you get L/C information from client in futre. And also other fields data.




    Add Order Lines and add signature in sales order page.




    After save the form you will get a sales order number. And it will be save in quotation.




    Before Confirm the order, select PI Type as L/C. And you will find the L/C No. and Amend No/Date field. Add L/C with Sales Order.




    Import L/C informaton in L/C form.




    L/C No connect with that sales order.




    Confirm Sales Order after add L/C no in sales order page.




    Proforma Invoice PDF format.




    After Confirm the Order you will find a delivery order with Delivery Order No. in Delivery Orders menubar.




    Sample of delivery orders.




    You can create partial deliery order. Just click on edit and input quantity in done field in Operations tab and validate that form.




    After click validate button this form will apear. From here you can create backorder if you want to back other quantites into your warehouse.
